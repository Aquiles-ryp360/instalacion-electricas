# Guia de edicion del proyecto domiciliario

Esta carpeta indica donde editar cada parte del proyecto. La estructura sigue el orden del proyecto guia `S.U. ARICATO_PHRA-SANDIA`, pero adaptada a una instalacion electrica interior de vivienda.

## Donde editar

| Necesitas editar | Archivo | Que completar |
|---|---|---|
| Datos generales de la casa | `../datos-vivienda.md` | Ubicacion, area, pisos, ambientes, suministro y empresa distribuidora |
| Memoria descriptiva | `../01-memoria-descriptiva/memoria-descriptiva.md` | Generalidades, objetivo, base legal, descripcion de vivienda y caracteristicas tecnicas |
| Cuadro de cargas | `../cargas-electricas.md` | Ambientes, equipos, potencias, circuitos y demanda |
| Calculos justificativos | `../02-calculos-justificativos/calculos-justificativos.md` | Corrientes, conductores, caida de tension, protecciones y puesta a tierra |
| Especificaciones tecnicas | `../03-especificaciones-tecnicas/especificaciones-materiales.md` | Conductores, tubos, tablero, interruptores, tomacorrientes, luminarias y puesta a tierra |
| Montaje | `../04-montaje/especificaciones-montaje.md` | Reglas de instalacion, canalizaciones, pruebas y seguridad |
| Cronograma | `../05-cronograma/cronograma.md` | Actividades y duracion |
| Metrados | `../06-metrados/metrado.md` | Cantidades de materiales |
| Planos | `../07-planos/README.md` | Lista de planos que debes dibujar o insertar |
| Calculadora HTML | `../../herramientas/calculadora-instalacion-casa.html` | Cargar ambientes y consumos para obtener cuadro de cargas preliminar |

## Planos que necesitas preparar

1. Plano de ubicacion de la vivienda.
2. Plano de arquitectura simple o croquis con ambientes.
3. Plano de alumbrado.
4. Plano de tomacorrientes.
5. Plano de circuitos y canalizaciones.
6. Diagrama unifilar del tablero general.
7. Cuadro de cargas del tablero.
8. Detalle de puesta a tierra.

Guarda los planos en `../07-planos/`. Si aun no los tienes, deja una captura, foto o croquis escaneado en esa carpeta y actualiza la tabla de planos.

## Regla importante

El proyecto guia es de red primaria. Para tu casa solo se toma como modelo de orden y presentacion. La normativa principal para el diseno interior es:

- Codigo Nacional de Electricidad - Utilizacion.
- RNE EM.010 Instalaciones Electricas Interiores.
