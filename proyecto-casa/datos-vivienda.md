# Datos de la vivienda

Datos base definidos para el caso de estudio del mini proyecto.

| Dato | Valor |
|---|---|
| Ubicacion | Jr. Lima N/S, distrito de Capachica, provincia de Puno, departamento de Puno |
| Tipo de vivienda | Unifamiliar |
| Numero de pisos | 3 pisos |
| Area aproximada | 5 m x 8 m = 40 m2 por piso |
| Ambientes | Sala, cocina, dormitorios, bano, lavanderia y circulaciones, segun levantamiento arquitectonico |
| Tension de suministro | 220 V, 60 Hz, monofasico, sujeto a verificacion con la empresa distribuidora |
| Empresa distribuidora | Electro Puno S.A.A., sujeto a verificacion local |
| Integrantes del grupo | Renzo Gabriel Mamani Galindo |
| Tablero general | Tablero general de distribucion empotrado, ubicado en zona accesible del primer piso, proximo al ingreso principal y al punto de alimentacion |

## Ambientes

| Ambiente | Alumbrado | Tomacorrientes | Cargas especiales | Observaciones |
|---|---:|---:|---|---|
| Sala | Si | Si | No | Considerar alumbrado general y tomacorrientes de uso comun |
| Cocina | Si | Si | Si | Considerar tomacorrientes para pequenos artefactos y carga dedicada si corresponde |
| Dormitorios | Si | Si | No | Considerar puntos de alumbrado y tomacorrientes generales |
| Bano | Si | Si | No | Tomacorriente protegido y ubicado fuera de zona humeda directa |
| Lavanderia | Si | Si | Si | Considerar lavadora como carga de uso especifico |
| Circulaciones/escalera | Si | No | No | Considerar alumbrado de paso para los tres pisos |
