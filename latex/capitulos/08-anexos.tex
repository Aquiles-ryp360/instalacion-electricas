\section*{Anexo C: Proyecto guia}

Se uso como referencia de estructura el proyecto \textit{S.U. ARICATO\_PHRA-SANDIA}, enviado como material del curso. Dicho proyecto corresponde a red primaria; por tanto, se toma como molde de presentacion y no como criterio tecnico directo para una instalacion interior domiciliaria.
