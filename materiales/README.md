# Materiales del curso

Esta carpeta guarda los materiales seleccionados desde la carpeta de Google Drive del curso.

## Organizacion

| Carpeta | Contenido | Uso sugerido |
|---|---|---|
| `normas/` | PDFs del Codigo Nacional de Electricidad - Utilizacion y EM.010 | Sustento normativo principal |
| `proyecto-guia-red-primaria/` | Proyecto S.U. ARICATO_PHRA-SANDIA enviado por el ingeniero | Referencia de estructura, memoria, calculos, metrados, especificaciones y planos |
| `fotos-clase/` | Fotos y capturas compartidas para la primera unidad | Evidencia o apuntes de clase |
| `otros-referencia/` | PDFs complementarios de presentaciones, valorizacion y liquidacion | Consulta secundaria |

## Criterio de seleccion

Se incluyeron archivos utiles para el mini proyecto y se evitaron duplicados o temporales:

- No se subio `S.U. ARICATO_PHRA-SANDIA.rar` porque duplica el proyecto descomprimido.
- No se subieron archivos temporales como `.bak`, `.dwl`, `.dwl2` o `plot.log`.
- El proyecto S.U. ARICATO_PHRA-SANDIA es de red primaria; debe usarse como guia de formato y metodologia, no como criterio directo para una instalacion interior domiciliaria.

## Como usar el proyecto guia

1. Revisar `aspectos-generales/` para ver orden, indice y resumen.
2. Revisar `capitulo-i-memoria/` como modelo de memoria descriptiva.
3. Revisar `capitulo-ii-calculos/` para tomar referencia de tablas, hojas de calculo y diagramas.
4. Revisar `capitulo-iii-especificaciones/` para redactar especificaciones tecnicas.
5. Revisar `capitulo-vi-metrados/` para preparar una tabla de metrados del mini proyecto.
6. Revisar `capitulo-vii-planos/` solo como referencia grafica.
